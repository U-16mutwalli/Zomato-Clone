# Zomato Clone

A responsive web application inspired by Zomato, designed to help users discover the best food and dining spots. The application features a dynamic search bar, restaurant listings, user login/signup options, and a visually appealing UI.

## Features

- **Search Functionality**: Allows users to search for restaurants and food by location or name.
- **User Authentication**: Includes login and signup options for users.
- **Responsive Design**: Optimized for mobile, tablet, and desktop screens.
- **Restaurant Listings**: Displays top trending spots, newly opened places, romantic dining, and best rooftop restaurants.
- **Accessibility**: Utilizes `tabindex` for enhanced keyboard navigation and accessibility.
- **Interactive UI**: Hover effects, drop-downs, and clickable buttons for improved user experience.
- **Zomato App Promotion**: Includes a section to download the Zomato app.

* Technologies Used

- **Frontend**: HTML, CSS, JavaScript, ReactJS
- **Styling**: CSS for custom styles, responsiveness, and hover effects.
- **Routing**: React Router for navigation between pages.
- **Icons**: FontAwesome icons for UI elements like search, login, and app promotion.

## Project Setup

1. Clone the repository:
  bash
   git clone https://github.com/your-username/zomato-clone.git
 
2. Navigate into the project directory:
  bash
   cd zomato-clone
  
3. Install dependencies:
  bash
   npm install
  
4. Start the development server:
  bash
   npm start
